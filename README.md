# CineVote

> Built at SYNthesis — CNSA's Premier Hackathon (April 19, 2026)

A real-time group movie picker. Create a room, invite friends, rate movies, and let the crowd decide.

## How It Works

1. **Create a room** — get a shareable room code
2. **Everyone joins** — share the code, no account needed
3. **Rate movies** — each user rates movies 1–5 stars
4. **See the results** — ratings are averaged across all users and displayed as a ranked list
5. **Top rated wins** — the group's pick is clear

## Features

- Room creation with unique join codes
- Real-time multi-user rating
- Averaged scores across all participants
- Ranked results leaderboard
